Pokedex

Mon site web Pokemon est un projet personnel que j'ai développé afin de créer une base de données en ligne de tous les Pokemon de la franchise. Il est réalisé en PHP avec le framework Laravel, qui offre de nombreux outils et fonctionnalités pour faciliter le développement d'applications web. J'ai utilisé Blade, un moteur de template qui permet de créer des vues en utilisant du code PHP simple et lisible, ainsi que Breeze, un ensemble d'outils de gestion de l'interface utilisateur qui rend la création de pages web plus rapide et plus simple. Enfin, j'ai également utilisé Eloquent, le système de gestion de données de Laravel, qui permet de manipuler facilement les données de la base de données. Mon site web est donc une application complète et performante qui permet aux fans de Pokemon de découvrir tous les détails de leurs personnages favoris.
