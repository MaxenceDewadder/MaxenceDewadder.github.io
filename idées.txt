Mon logiciel d'automatisation de tâches sur jeu vidéo est un projet personnel que j'ai développé pour améliorer mes performances dans les jeux vidéo que je joue. Il est écrit en python et utilise une technologie de reconnaissance d'image et de texture pour détecter les éléments de l'interface du jeu et exécuter les actions souhaitées. En outre, j'ai également intégré Android Debug Bridge (ADB), un outil de développement pour Android, afin de pouvoir contrôler l'appareil et exécuter des commandes sur celui-ci. Grâce à ces technologies, mon logiciel est capable de jouer de manière autonome et de progresser dans le jeu sans intervention humaine. Je suis très fier de ce projet et j'ai hâte de continuer à l'améliorer et à le rendre encore plus performant.

Mon site web Pokemon est un projet personnel que j'ai développé afin de créer une base de données en ligne de tous les Pokemon de la franchise. Il est réalisé en PHP avec le framework Laravel, qui offre de nombreux outils et fonctionnalités pour faciliter le développement d'applications web. J'ai utilisé Blade, un moteur de template qui permet de créer des vues en utilisant du code PHP simple et lisible, ainsi que Breeze, un ensemble d'outils de gestion de l'interface utilisateur qui rend la création de pages web plus rapide et plus simple. Enfin, j'ai également utilisé Eloquent, le système de gestion de données de Laravel, qui permet de manipuler facilement les données de la base de données. Mon site web est donc une application complète et performante qui permet aux fans de Pokemon de découvrir tous les détails de leurs personnages favoris.

Mon projet de groupe consistait en la création d'un générateur d'image de fractal en Java. Un fractal est une figure géométrique complexe et répétitive qui peut être générée par un algorithme mathématique. Pour réaliser notre projet, nous avons dû travailler avec les nombres complexes, qui sont des nombres permettant de représenter un point dans un plan complexe. Nous avons utilisé ces nombres pour créer des algorithmes qui permettaient de générer des fractals de manière algorithme. Le résultat final était une application capable de générer des images de fractals de manière interactive, en permettant à l'utilisateur de définir les paramètres de l'algorithme et de visualiser les résultats en temps réel. Ce projet a été une expérience passionnante pour moi et m'a permis de découvrir de nouvelles techniques de programmation et de travailler en équipe.
